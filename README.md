# Zied Alimi Portfolio

A personal portfolio website built to present my projects, technical skills, experience, and professional profile.

## Project Status

This project is currently under development.

The first version was created using HTML, CSS, and JavaScript. The project will gradually be migrated to React and expanded with modern development, deployment, and automation practices.

## Final Project Goal

The goal is to build a professional, responsive, and production-ready developer portfolio that demonstrates both frontend development and deployment skills.

The final version is planned to include:

- Responsive and mobile-friendly design
    
- React and modern JavaScript
    
- Projects and experience sections
    
- Downloadable résumé
    
- Contact form
    
- GitHub and LinkedIn integration
    
- SEO and accessibility improvements
    
- Automated deployment
    
- Custom domain
    
- CI/CD workflow
    
- Performance monitoring
    

## Planned Development Stages

### Phase 1 — Initial Portfolio

- Build the first version with HTML, CSS, and JavaScript
    
- Create the main sections
    
- Make the website responsive
    
- Deploy a test version
    

### Phase 2 — React Migration

- Rebuild the interface using React
    
- Create reusable components
    
- Improve project structure
    
- Move portfolio content into structured data files
    

### Phase 3 — Production Improvements

- Add testing and code-quality checks
    
- Configure GitHub Actions
    
- Improve SEO, accessibility, and performance
    
- Add a contact form and analytics
    

### Phase 4 — Final Deployment

- Connect a custom domain
    
- Configure HTTPS
    
- Add monitoring
    
- Publish the final production version
